SIGNIFY

Signify uses machine learning to recognize and interpret sign language gestures,and to make online browsing and navigation more accessible making it possible for people who have speaking or hearing impairment to communicate more easily with the system. Below are the steps taken for Signify to interpret American Sign language.

OpenCV Pipeline
OpenCV is an open-source computer vision library that provides a set of powerful tools for image and video processing, including image recognition, object detection, and feature extraction. The pipeline typically involves capturing or inputting visual data, applying pre-processing steps such as noise reduction and image resizing, performing object detection or tracking, and finally outputting the results.
Tensorflow CNN
TensorFlow is an open-source machine learning framework that provides a set of tools for building and training artificial neural networks. In the sign language detection system, TensorFlow is often used to implement convolutional neural networks (CNNs), so that the system can process image data effectively.
Pyttsx3 Module
Pyttsx3 is a Python library that provides a cross-platform text-to-speech (TTS) engine. In the sign language detection systems, Pyttsx3 can be used to provide a spoken output of the recognized sign language gesture.Once the gesture is recognized by the system, the corresponding text or message is generated, which is then passed to Pyttsx3 to generate an audio output of the message. Pyttsx3 generates a spoken output of the recognized gesture, which can be played through a speaker or headphone to provide feedback to the user.
Google Teachable Machine
Google's Teachable Machine is an online tool that allows users to create machine learning models. It is a web-based application that uses a browser's webcam to collect visual or audio data, which can be used to train a model. Teachable Machine uses TensorFlow.js, a JavaScript library that enables machine learning in web browsers.It is used in a sign language detection system by training the model to recognize and classify different sign language gestures.
